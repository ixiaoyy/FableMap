# FableMap Character Visual Style Recipes

> User-contributed reusable image-prompt recipes for NPC/character material ideation. These are text recipes only; no referenced example images are stored here.

## How to use

- Replace `[在此处替换为您想要生成的主体内容]` with an **original** FableMap tavern NPC, tavern host, owner avatar, or authorized campaign subject.
- For FableMap runtime assets, add tavern cues: bar counter, mugs, shelves, lanterns, menu board, bottles, map-table, cyber terminal glow, door signage, or equivalent.
- Do not ship prompts that rely on specific copyrighted characters, franchise marks, brand logos, or living-artist imitation unless the user owns/has authorization and the final asset path remains compliant with project rules.
- Preserve the whole recipe when the user provides a tested prompt; only adapt subject, palette, layout ratio, and tavern-specific props unless asked to rewrite.

## Style selection heuristics

- **Do not pile up all styles.** Pick one strong recipe and at most one supporting texture system.
- **Color fatigue is real.** Y2K electric blue/orange and high-density collage look impressive but can become noisy if reused too often.
- **Density needs contrast.** If the subject has dense fragments, keep large negative space; if the background is dense, simplify the character silhouette.
- **Prompt structure to preserve:** subject pose → composition → style lineage → palette → medium/texture → symbolic/digital artifacts → constraints.
