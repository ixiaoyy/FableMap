# FableMap 文化诠释框架 v0.1

## 目标

这份文档用于定义：FableMap 如何把现实世界中的地点、设施和城市秩序，翻译为带有幻想感、黑色幽默和社会隐喻的平行世界语义。

它不是单纯的“改名表”，而是整个项目的解释学边界。

## 设计原则

### 1. 不只换皮，要保留现实功能的影子

现实地点进入异世界后，不能只变成一个更酷的名字，还要保留它在现实中的社会功能、控制关系与情绪位置。

例如：

- 警察局不只是魔法塔，更是秩序、威慑和程序正义幻觉的节点
- 便利店不只是杂货铺，更是疲惫城市的微型续命装置
- 红绿灯不只是信号器，而是城市秩序被执行的微型仪式

### 2. 黑色幽默不是嘲笑，而是揭示结构

FableMap 的讽刺对象不是个体，而是：

- 资源分配
- 权力装置
- 都市劳动
- 现代孤独
- 被包装成温柔的高压秩序

### 3. 同一地点应同时允许“锋利”与“柔软”两种读法

FableMap 既服务于权力/政治/动荡叙事，也服务于审美/记忆/治愈叙事。

因此每个地点的解释，最好同时预留：

- `satire_hook`
- `emotion_hook`
- `visual_hint`

### 4. 解释必须可追溯到 Schema

文化诠释不应停留在散文层，应尽量能落到现有世界结构里：

- `pois[*].fantasy_type`
- `pois[*].satire_hook`
- `pois[*].emotion_hook`
- `pois[*].faction_alignment`
- `region.theme`
- `region.satire_profile`
- `region.vibe_profile`

## 诠释层级

建议把一次地点翻译拆成四层：

1. **现实功能层**：这个地点在现实中做什么
2. **社会结构层**：它维持了什么秩序、焦虑、依赖或欲望
3. **异世界语义层**：它在平行世界里变成什么机构、遗迹或节点
4. **体验层**：它给玩家带来什么叙事、审美和情绪体验

## 高优先级诠释词条

### 警察局 / 执法设施

- 现实功能：执行秩序、威慑风险、处理冲突
- 异世界语义：`judgement_tower` / `order_bureau`
- 讽刺方向：秩序总以保护为名，但也可能生产恐惧
- 情绪方向：安全感与压迫感并存
- 可关联字段：
  - `fantasy_type=judgement_tower`
  - `faction_alignment=order_bureau`
  - `satire_hook=Peace is carefully audited here.`

### 医院 / 诊疗设施

- 现实功能：修复、诊断、筛查、等待
- 异世界语义：`healing_sanctum` / `plague_research_house`
- 讽刺方向：治愈既是关怀，也是成本与门槛
- 情绪方向：脆弱、希望、被照看、被编号

### 便利店 / 小型零售

- 现实功能：即时补给、夜间续航、最低限度的便利
- 异世界语义：`supply_outpost`
- 讽刺方向：一切都可购买，但不是人人都买得起安稳
- 情绪方向：深夜生存感、熟悉感、城市余温

### 咖啡馆 / 茶饮店

- 现实功能：停留、社交、发呆、短暂自我修复
- 异世界语义：`ember_parlor`
- 讽刺方向：温柔空间往往由消费资格维持
- 情绪方向：交谈、等待、含混的安慰

### 学校 / 教育设施

- 现实功能：训练、筛选、社会化、未来分配
- 异世界语义：`lore_academy`
- 讽刺方向：知识可能是启蒙，也可能是阶层门票
- 情绪方向：规训、期待、青春残响

### 公园 / 绿地

- 现实功能：缓冲、休憩、城市呼吸口
- 异世界语义：`whispering_grove`
- 讽刺方向：自然常被规划成可控的疗愈装置
- 情绪方向：放松、散步、记忆寄存

### 办公楼 / 商务区

- 现实功能：劳动组织、等级秩序、价值提炼
- 异世界语义：`ledger_spire` / `contract_citadel`
- 讽刺方向：效率语言掩盖了消耗、替代和悬置生活
- 情绪方向：紧绷、空调化、匿名协作疲劳

### 老旧公寓 / 住宅区

- 现实功能：居住、维持日常、家庭与孤独并存
- 异世界语义：`stacked_sanctuary` / `echo_residence`
- 讽刺方向：家的意义常被价格、密度和通勤吞掉
- 情绪方向：私密、重复、潮湿温度、生活残响

### 红绿灯 / 路口控制节点

- 现实功能：维持交通秩序、切分通行权
- 异世界语义：`order_beacon` / `threshold_sigil`
- 讽刺方向：秩序以秒为单位分配耐心和优先级
- 情绪方向：等待、犹豫、城市拍点

### 垃圾桶 / 回收设施

- 现实功能：收纳废弃物、维持表面整洁
- 异世界语义：`memory_sink` / `discard_shrine`
- 讽刺方向：城市靠持续忘记自己才能看起来干净
- 情绪方向：丢弃、轻微羞耻、生活证据被擦除

### 快餐店

- 现实功能：快速补充热量、标准化就餐
- 异世界语义：`industrial_feeding_station`
- 讽刺方向：被优化的不只是食物，还有休息时间和人的耐心
- 情绪方向：疲惫、便捷、被照顾的幻觉

## 区域级诠释

FableMap 不应只解释单个 POI，还要解释一整片区域如何构成共同气质。

区域级诠释重点看：

- 土地用途主导关系
- POI 密度与组合方式
- 道路宽度与通行结构
- 公共设施与消费设施的比例
- 白天/夜晚的功能反差

输出时可沉淀为：

- `region.theme`
- `region.atmosphere`
- `region.satire_profile`
- `region.class_tone`
- `region.comfort_level`

## 需要避免的坏味道

以下内容应尽量避免：

- 只做幼稚直译：如“医院 -> 治疗屋”但没有社会语义
- 只有黑暗没有温度：让整个世界只剩控诉与压抑
- 只有美术滤镜没有结构：地点看起来美，但没有叙事理由
- 只追求猎奇：为了怪诞而怪诞，失去现实映射张力

## 与未来系统的关系

这份文档会为后续系统提供基础语义边界：

- **Street as a Sentence**：决定地点序列如何形成一致气质
- **Causal Ripple Algorithm**：决定一个节点的社会功能为何会向外传播影响
- **Aesthetic Kernel**：决定同一地点在不同镜头下如何保持解释一致性
- **文化诠释词典 JSON**：未来可再做程序可消费版本
