# FableMap 长期体验方向

## 定位

FableMap 的长期目标不是儿童式奇幻换皮，而是一个能够同时承载**黑暗社会结构**与**审美情感表达**的现实空间 RPG。

更接近的参考气质是：

- 《极乐迪斯科》的社会观察
- 《黑客帝国》的现实错位感
- 克苏鲁式地点谜团
- 脏冷赛博或暗黑哥特的视觉气质
- 《纪念碑谷》式审美凝视
- 《光遇》式情感连接与静谧体验

## 双主线结构

FableMap 的长期体验不应该只有一种语气，而应该形成两条并行的主线：

### 主线 A：权力、讽刺与世界状态

这条线负责黑暗、复杂、社会化和成人感。

### 主线 B：审美、情感与治愈收集

这条线负责分享欲、私密感、自我表达、收藏欲与情绪价值。

两条主线共用同一套真实地图骨架，但会使用不同的叙事镜头与交互回路。

## 主线 A：三个长期核心维度

### 1. 阵营与权力

真实城市不是中性的，区域之间总带有资源、阶层和规则差异。FableMap 的长期版本要把这种结构转成可玩的系统：

- 高消费 / 高房价区域 -> 排外、秩序强、资源集中的上层区
- 老旧工业区 / 城中村 -> 灰色交易、叛军网络、地下工坊
- 商业中心 -> 资本派系、物流路线、舆论争夺中心

玩家不是旁观者，而是阵营结构的一部分。通勤路线、停留地点、完成事件都可能影响区域控制力、资源流向和声望关系。

### 2. 现实生活的数值化与讽刺

FableMap 不应只做浪漫幻想，也应该具备成年人能共鸣的黑色幽默：

- 健身房 -> 肉体改造实验室
- 写字楼 -> 灵魂压榨磨盘
- 便利店 -> 中立补给站
- 医院 -> 圣疗机构 / 生物伦理边界地带

系统不是简单贴标签，而是把现实职业、消费、通勤、绩效和城市压力转成带讽刺感的互动文本与任务逻辑。

### 3. 隐秘历史与地点谜团

城市每一块区域都有真实历史残留。长期版本会把历史事件、城市传闻与真实地点背景转成世界中的“回声”：

- 某地过去的灾难、新闻事件、产业更替
- 某栋建筑的用途变迁
- 某街区特有的城市记忆

这些信息不会直接平铺给玩家，而会被碎片化埋入场景、NPC 对话、任务物品与区域异常中。

## 主线 B：三个长期核心维度

### 1. 审美重构与 Vibe Camera

现实世界中的同一条街道，不必永远只有一种“赛博废土”解释。长期版本中，系统应支持多种风格化重构：

- 吉卜力式清新城镇
- 王家卫式霓虹怀旧都市
- 莫奈式印象派花园
- 暗黑哥特或脏冷赛博夜景

这不仅是视觉滤镜，而是一种能改变地点叙事、氛围音色和可收集对象分布的世界镜头。

### 2. 情感锚点与 Secret Garden

地点不只是地理节点，也可以是情绪容器：

- 某张长椅保留一段匿名心事
- 某个街角允许用户埋下只对好友可见的虚拟信物
- 某家店铺会随着天气或时间出现特殊的情绪文本

这种设计让现实世界获得“可被记住、可被表达、可被温柔占据”的第二层意义。

### 3. 治愈收集与 Mirror Home

长期版本可加入更轻盈但高粘性的收集回路：

- 现实中的奶茶店生成饮品系精灵
- 下雨天的公交站生成天气系精灵
- 公园、花店、书店、车站产出不同装饰碎片

玩家可以把这些收集物带回一个私人空间，即 Mirror Home，用于展示、摆放和装扮自己在现实中走出来的平行记忆。

## 动态世界信号

长期版本的关键升级，是让世界不是静态地图，而是会受到现实节奏影响。

候选信号包括：

- 人流量
- 交通拥堵
- 营业时间变化
- 区域消费水平
- 天气与昼夜

这些信号可以被转译为：

- 社会动荡程度
- 贸易活跃度
- 异常事件概率
- 阵营巡逻强度
- 谜团触发门槛
- 风格滤镜强度
- 稀有精灵出现概率
- 某些情感锚点的开放条件

## 对产品规划的影响

### 对 v0.1 的影响

不改变 v0.1 的核心目标。第一版仍然只做：

> 坐标 -> 世界 JSON -> 一致记忆 -> Godot 可探索 Demo

### 对 v0.2+ 的影响

中期将逐步增加以下系统：

- 区域阵营与控制力
- 虚拟经济与资源流
- 黑色幽默文案层与社会讽刺模板
- 历史回声与谜团链路
- 动态事件与地区扰动模型
- Vibe Camera 与多风格世界镜头
- Secret Garden 与地点记忆锚点
- Urban Sprite Collector 与 Mirror Home

## 对开发者的吸引力

这个方向的意义在于：FableMap 不再只是“地图变奇幻”，而是开始触碰一个更难、更有野心的问题：

> 如何让 AI 把现实城市的社会结构、历史残留、审美情绪与私人记忆，一起转译成一个可探索、可参与、可演化的第二世界。
