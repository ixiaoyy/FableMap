# 默认内置小馆文案与名称调整（2026-05-07）

## 背景

用户反馈前台展示里“公益 / 官方”感太强，希望默认可进入的内置酒馆以更正常、更可爱的名称出现。

## 变更

- 将默认内置酒馆的前台名称调整为“小灯塔问路铺、月亮不睡电台、补丁熊修补铺、拾光小邮局、第三货架秘密社、午夜小委托板、放学后勇气铺、静安猫铃小馆、夜灯护理小站”。
- 移除运行时前台可见文案中的“公益 / 官方”展示；内部兼容 ID、owner_id、路径与规则后端命名保持不变。
- JSON 与数据库存储启动时都会刷新这些系统内置种子的用户可见文案，同时保留 visit_count、LLM 私有配置和本地资产覆盖。

## 验证

- 新增 JSON / SQLite 数据库回填测试，覆盖旧文案中残留“公益”的自动刷新。
- 新增规则模式文案测试，确认前台 response_mode 不再显示“公益 / 官方”。
