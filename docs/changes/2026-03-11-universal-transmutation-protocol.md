# 变更说明：万物转义协议文档接入

## 本次改动

- 新增 `docs/UNIVERSAL_TRANSMUTATION_PROTOCOL.md`
- 更新 `README.md` 文档导航，加入协议级入口
- 更新 `docs/AI_SHARED_TASKLIST.md`，补充与协议拆解相关的可认领任务

## 改动目的

把此前只存在于对话中的高层世界设计思路，沉淀成仓库内可引用的统一基线，避免后续 AI 参与者在以下方向各写各的：

- 地点转义
- 双轨镜头
- 动态扰动
- 玩家写回
- 历史深度 / 尘封之眼

## 核心沉淀内容

- 现实骨架、基础原型、镜头坍缩、动态扰动、玩家写回与历史深度的五层协议
- “同一地点如何在不同镜头下坍缩成不同身份”的 Dual-Track Mapping 思路
- “现实信号如何变成超自然状态”的扰动解释层
- “玩家与历史如何给城市写回厚度”的权限与证据观念

## 影响范围

- 只影响设计文档、协作入口与任务拆分
- 不影响现有代码运行行为
- 不修改现有 schema 字段与 CLI 协议

## 验证

- 交叉检查与 `WORLD_SCHEMA`、`CULTURAL_INTERPRETATION`、`DISTURBANCE_MODEL`、`WEB_2D_SPIRIT_VIEW` 的口径一致性
- `git diff --check`
