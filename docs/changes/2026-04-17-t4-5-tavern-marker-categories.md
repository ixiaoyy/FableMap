# T4.5 标记分类显示

## 背景

T4.4 已经让地图发现页可以浏览和搜索空间。T4.5 继续补齐访问类型识别，让访客在点击前就能区分公开、密码和私人入口。

## 改动

- 地图空间 marker 按 `public` / `password` / `private` 使用不同颜色、短标签和入口图标。
- 地图叠加层新增当前可见空间分类图例，展示每类数量。
- 空间发现列表改用统一的访问类型 chip。
- 发现筛选入口加入私人分类，便于 owner 视图或未来授权发现复用。
- Map adapter 销毁时同步清理空间 marker，避免地图重建后残留。

## 验证

- `npm run build`
- `git diff --check`
