# 变更说明：项目描述与最终世界愿景补强

## 本次改动

- 更新 `README.md` 的项目简介与项目气质延展说明
- 新增对 FableMap 最终世界效果的具象描述
- 更新 `index.html` 首页 Hero 文案，中英文口径同步

## 改动目的

让第一次进入仓库或首页的人，不只知道 FableMap 会“把现实地图转成幻想世界”，还能够更具体地理解：

- 它最终不是一张静态漂亮地图
- 而是一个覆盖在真实城市上的第二世界
- 这个世界同时承载黑暗社会寓言与治愈情感表达
- 玩家不只是浏览者，也会是留下痕迹与参与改写的人

## 影响范围

- 只影响项目描述、产品定位与首页展示文案
- 不修改运行时代码行为
- 不改变协议、Schema、字段或测试边界

## 验证

- 文档间口径一致性检查
- `git diff --check`

## 备注

本次文案明确区分了两层：

1. 当前已经存在的 nearby / page / bundle / Web-2D 入口方向
