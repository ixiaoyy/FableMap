## 菜园酒馆 (Garden Tavern) MVP 设计

### 核心定位

- 作为特殊酒馆类型存在（宠物酒馆同理）
- NPC "农场管家"主导对话和引导
- 作物种植/交易作为副玩法，不是独立农场游戏
- 所有经济活动在酒馆内闭环，不做跨酒馆交易

### 功能模块设计

根据截图参考，菜园酒馆包含以下功能模块：

| 模块 | 功能 | NPC 关联 |
|------|------|----------|
| 农场 | 种植、浇水、收获作物 | 农场管家指导 |
| 仓库 | 存放作物、库存管理 | 管家播报库存 |
| 交易所 | 作物定价、交易 | NPC 管家主持 |
| 偷菜 | 顺走其他访客作物 | 管家通知被偷 |
| 排行榜 | 酒馆内产量排名 | 不做跨酒馆 |
| 好友农场 | 访客作物互访 | 酒馆内闭环 |

### 经济系统设计

**1. 货币系统**

- 使用"黑钻"作为游戏内货币
- 访客通过以下方式获取：
  - 收获作物出售
  - 完成 NPC 委托任务
  - 每日登录奖励
- 货币用途：
  - 购买种子
  - 购买道具（肥料加速等）
  - 参与酒馆活动

**2. 作物系统**

参考作物列表（含价格示例）：

| 作物 | 单价范围 | 生长周期 | 价值等级 |
|------|----------|----------|----------|
| 蓝莓 | $0.08-0.12 | 短 | 低 |
| 玉米 | $0.03-0.05 | 中 | 低 |
| 草莓 | $0.10-0.15 | 中 | 中 |
| 胡萝卜 | $0.25左右 | 短 | 中 |
| 西红柿 | $0.05-0.08 | 中 | 低 |
| 南瓜 | $0.12-0.18 | 长 | 中 |
| 橙子 | $0.03-0.06 | 中 | 低 |
| 芒果 | $0.02左右 | 中 | 低 |
| 西瓜 | $1.00（锚定价）| 长 | 高 |

**3. 交易所机制**

参考截图中的交易所设计：

- **锚定价格**：西瓜固定为 $1，作为价格基准
- **价格浮动**：交易量影响作物价格
  - 卖出量增加 → 价格下跌
  - 买入量增加 → 价格上涨
- **NPC 播报**：农场管家在对话中播报价格动态
  - "今日蓝莓价格上涨 5%"
  - "西瓜价格稳定，建议观望"
- **价格趋势**：显示涨跌幅（+/% - /%）

**4. 交易所对话示例**

```
访客：今天价格怎么样？
管家：今日行情——蓝莓涨了3%，西瓜锚定在$1，芒果小幅下跌。
      胡萝卜需求量高，可以考虑种植。

访客：我要卖西瓜
管家：当前西瓜价格 $1，已锚定。确定要出售 5 个西瓜吗？
      预计收益：$5 黑钻
```

### 偷菜机制设计

**核心规则**：

1. **偷菜入口**：访客可进入"好友农场"查看其他访客的作物
2. **触发条件**：其他访客有成熟作物
3. **每日限制**：每个访客每日偷菜次数有限（建议 3-5 次）
4. **被发现机制**：
   - 被偷访客收到 NPC 通知
   - 管家会告知："你的一株蓝莓被偷了，但没关系，明天会重新长出"

**NPC 通知示例**：

```
管家：提醒你，刚才有访客顺走了你的 2 个草莓。
      别担心，你的草莓田明天就会重新成熟。
```

### 仓库系统

**功能**：

- 存放已收获但未出售的作物
- 查看库存数量
- 一键出售到交易所

**NPC 交互**：

```
管家：你的仓库里有 10 个蓝莓、5 个玉米、2 个西瓜。
      要现在卖掉吗？西瓜目前价格稳定。
```

## 用户确认的决策

### 宠物酒馆 (Pet Tavern)

**定位**：以养宠互动为核心的酒馆

**NPC 角色**：宠物管家（可配置为猫/狗/精灵等形象）

**核心玩法**：
- 宠物有独立性格（好奇型/傲娇型/治愈型）
- 访客可以与宠物对话（简化版 LLM）
- 宠物会记住访客的喂食/互动次数
- 访客离开时宠物留下"宠物便签"给店主
- 宠物可对访客与 NPC 的对话做出旁观反应

**复用**：
- 使用现有 NPC 角色系统
- 宠物对话使用简化版 prompt
- 宠物状态可复用 `VisitorState`

### 菜园酒馆 (Garden Tavern)

**定位**：以种菜偷菜为核心的酒馆

**NPC 角色**：农场管家

**核心玩法**：
- 访客可以种植、浇水、施肥
- NPC（农场管家）通过对话响应种植动作
- 作物成熟时 NPC 通知访客
- "偷菜"机制：可顺走其他访客的成熟作物（每日有限次数）
- 被偷时 NPC 告知并生成补偿提示

**防沉迷**：
- 每日偷菜次数限制
- 作物生长周期可配置
- 不做付费加速

**复用**：
- 作物状态可作为 `WorldInfoEntry` 或私有数据
- NPC 管家使用现有角色系统

### 游戏工坊酒馆 (Game Workshop Tavern)

**定位**：4399 风格小游戏集成，NPC 全程参与

**NPC 角色**：游戏解说

**核心玩法**：
- 提供小游戏入口列表（2048、问答挑战、猜谜等）
- NPC 作为游戏解说或对手
- 游戏结束后 NPC 评价表现
- 成就可解锁 NPC 特殊对话

**技术约束**：
- iframe 沙箱隔离
- 只集成有明确开源许可的游戏
- 复用已有 2048 方案

**复用**：
- 游戏进度作为访客私有数据
- NPC 解说使用现有 LLM 接口

### 抽卡酒馆 (Gacha Tavern)

## 特殊酒馆类型的薄层实现

参考 `05-06-special-tavern-type-thin-layer` 的设计：

```typescript
type SpecialTavernType = 
  | "pet"        // 宠物酒馆
  | "garden"     // 菜园酒馆
  | "game"       // 游戏工坊酒馆
  | "gacha"      // 抽卡酒馆
```

每个类型决定：
- 默认 layout_style
- 默认 NPC 角色模板
- 默认功能卡片
- 默认 UI 主题
